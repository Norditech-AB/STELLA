## Quick Start

<details><summary>How to Set Up STELLA in Minutes</summary>
<p>

1. **Install Dependencies**: `pip install -r requirements.txt`
2. **Initialize**: Run `python run.py` from `/app`.
3. **Explore**: Dive into `/app/agents/DefaultAgents`.
